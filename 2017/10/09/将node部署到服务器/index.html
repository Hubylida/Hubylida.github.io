将node部署到服务器

2017-10-09

环境说明

  • 本地操作环境Mac OSX
  • 本地 node + git + pm2
  • 美团云主机：centos7.2 64位
  • 项目 Node.js + express.js

本地连接服务器

终端ssh无密登陆

在云服务器上添加密钥，把本地的公钥id_rsa.pub的内容复制。

ssh-keyscan -t rsa github.com >> ~/.ssh/known_hosts

本地连接服务器

ssh root@101.236.29.56

远端服务器配置环境

更新 yum

更新完毕后，安装 node

yum install nodejs -y

安装 npm

yum install npm -y

安装 pm2

yum install pm2 -g

安装 git

yum install git -y

查看版本号,确认安装成功

node -v, npm -v, pm2 -v

github建立连接

登录云服务器，生成ssh-key。然后将云服务器的ssh-key加到自己的github中。
在本地执行同样的操作。

在本地目录，调出终端。把项目更新到GitHub上。

git remote add origin git@github.com:Hubylida/FOCUS.git

git add .

git commit -m"new" 

git push orign master

本地安装好pm2后，在本地项目，调出终端。

pm2 ecosystem

会生成一个ecosystem.config.js(版本原因，也可能是ecosystem.json)文件，打开该文件进行配置。修改script配置为node项目启动文件。

放上我的配置文件

module.exports = {

apps : [

// First application
{
  name      : 'FOCUS',
  script    : 'app.js',
  watch     : true,
  env: {
    COMMON_VARIABLE: 'true'
  },
  env_production : {
    NODE_ENV: 'production'
  }
},

// Second application
{
  name      : 'WEB',
  script    : './bin/www'
}
  ],

  /**
  * Deployment section
  * http://pm2.keymetrics.io/docs/usage/deployment/
  */
  deploy : {
    production : {
      user : 'root',
      host : '101.236.29.56',
      ref  : 'origin/master',
      repo : 'git@github.com:Hubylida/FOCUS.git',
      path : '/var/root/www/development/FOCUS',
      'post-deploy' : 'npm install && pm2 reload ecosystem.config.js --env production'
    },
    dev : {
      user : 'root',
      host : '101.236.29.56',
      ref  : 'origin/dev',
      repo : 'git@github.com:Hubylida/FOCUS.git',
      path : '/var/www/development',
      'post-deploy' : 'npm install && pm2 reload ecosystem.config.js --env dev',
      env  : {
        NODE_ENV: 'dev'
      }
    }
  }
};

然后提交至GitHub上，以后改动配置文件都要重新提交。

git add .
git commit -m"update"
git push origin master

在云服务器端执行以下命令，将http://github.com加入known_hosts：

ssh-keyscan -t rsa github.com >> ~/.ssh/known_hosts

本地项目初始化到远程服务器端

pm2 deploy ecosystem.config.js production setup

一键部署代码到服务器

在本地项目，输入以下命令部署代码到服务器
pm2 deploy ecosystem.config.js production

更新代码 

pm2 deploy ecosystem.config.js production update

现在可以访问你的远端服务器的外网IP啦。自己绑定个域名到云服务器上美滋滋。
