自定义主页

我的主页

上一个版本用了有快四个月，昨晚在饱受babel的摧残之下，决定找回自信，于是就升级了主页。

这一版距上一版变化较大，首先最重要的是从GitHub移到了coding上来，这样加载速度快了许多。其次，时钟功能取消，我发现我看时间还是会看win10自带的时钟，所以直接取消了，这样排版也简洁了许多，还有就是把title改成LHHY代替了之前在内容区表达我对HH的爱（害羞）。输入框的背景改成了半透明，搜索框的实现也换了一种方案，最最重要的是实现了移动端的兼容，手机也可以打开了。

自定义步骤

点击查看源码

主页背景更改

body {
       background: url('http://ou2r9nim8.bkt.clouddn.com/4.jpg');
       width: 100%;
       height: 100%;
       overflow: hidden;
   }

找到background那一行，把背景图片更改为你想要的图片地址。

推荐一个动漫图片分享网站pixiv

键码设置

switch (event.keyCode) {
            case 102:
                location.href = "https://www.facebook.com/profile.php?id=100019094295164";
                break;
}

以一个作为例子，102是f的键码，location.href的值就是我的facebook的主页。
可根据你的喜好，更改键码和对应的网址。

主页上边的导航可以删除，也可以更改

<div id="nav">
    <a href="http://lhhyuan.coding.me/">我的博客</a>
    <a href="https://github.com/">我的GitHub</a>
    <a href="http://lhhyuan.coding.me/videoPlayer/play.html">Yeah视频</a>
</div>

href对应网址，a标签里是名字，你可以随意填。

更改好之后保存，把代码放到GitHub上或者coding上，用他们的pages服务，也可以自己买域名，去godaddy买最好，然后买虚拟主机，部署。

这样一个别人可以访问的网页就做好了，但是怎么把它实现为主页呢？

我用的是chrome的插件，在chrome的应用商店搜索新标签网站网址更改器，是个绿色的图标。

下载好了之后，点击插件，更改网址，然后点击save，当点击chrome的新标签页按钮时，新的标签页就会是你自定义的页面了。
如果想设置为浏览器的主页，可以在设置里面设置。
