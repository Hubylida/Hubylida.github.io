HTTP状态码

2017-09-18
Study

HTTP状态码

1XX (Informational)

接受的信息正在处理

2xx (Success)

请求正常处理完毕

200 OK 请求正常处理

204 No Content 请求处理成功但没有资源可以返回

206 Partial Content 对资源的一部分请求

3XX (Redirection)

重定向状态码
需要进行附加操作以完成请求

301 Moved Permanently 资源的URI已经更新，永久的重定向，需要更新自己的标签。

302 Fond 资源的URI临时定位到其他位置，暂时重定向。

303 See Other 表示请求的资源有另一个URI，问你要不要定位到另一个位置。

304 Not Modifie 资源找到了，但你不符合请求的要求，就是说，你没资格请求资源。

307 Temporary Redirect 临时重定向，但是不同于302 ，它不hi从post变为get。

4xx (Client Error)

客户端错误

400 Bad Request 你输错了

401 Unauthorized 需要验证身份

403 Forbidden 不许你访问那个资源，和303不同的是，303可以找到资源，但是不给你用，403直接就别想了。

404 Not Foound 服务器上没有你请求的资源

5xx (Server Error)

服务器端错误

500 Internet Server Error 服务器端在执行请求时发生了错误，也有可能是web应用存在bug或者是故障。

503 Server Unavailable 服务器正在停机维护或者处于超负载。
