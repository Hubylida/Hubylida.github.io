点击进度条效果和播放进度的实现

两层重合的div,背景色不一样,一条隐藏,一条显示,点击时,用e.offset确定点击的位置相对于显示的一条的位置,隐藏的一条的宽度就是 x, 再用x和w之商作为百分比和视频长度相乘就得到当前视频跳转的播放位置.同时要改变css显示出效果.

$('.v_ctl_du').on('click', function (e) {
    var w = $('#ctl_du_a').width(),
        x = e.offsetX;
    window.per = (x / w).toFixed(3);//全局变量
    var duration = player.duration;
    player.currentTime = (duration * window.per).toFixed(0);
    $('#ctl_du_b').css('width', x + 'px');
});

在视频播放时要时刻获取播放的进度,然后改变css,进度条就会更新样式,时间也会跟着改变.

$player.on('timeupdate', function () {
    var time = player.currentTime.toFixed(1);
    minutes = Math.floor((time / 60) % 60);
    seconds = Math.floor(time % 60);
    if (seconds < 10) {
        seconds = '0' + seconds;
    }
    $('#ctl_dur').text(minutes + ':' + seconds);
    var w = $('#ctl_du_a').width();
    if (player.duration) {
        window.per = (time / player.duration).toFixed(3);
    } else {
        window.per = 0;
    }
    $('#ctl_du_b').css('width', (window.per * w).toFixed(0) + 'px');
    if (player.ended) {
        pause();
    }
});

动画效果的实现
animate三个参数,第一个是样式的对象,第二个是时间,第三个是一个回掉函数.

(function () {
    var css = {
        'top': '49%'
    };
    $('.cloth_btn').animate(css, 400, rowBack);
    function rowBack() {
        if (css.top === "49%") {
            css.top = "51%";
        } else if (css.top === "51%") {
            css.top = "49%";
        }
        $('.cloth_btn').animate(css, 400, rowBack);
    }
})();
