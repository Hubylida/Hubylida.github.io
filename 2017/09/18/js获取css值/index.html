原生js获取css的值

2017-09-18
Study

getComputedStyle

使用getComputedStyle也可以获取元素的css属性值，第一个参数填元素，第二个参数填伪类。
这个方法是只读的，不能修改。
用这个方法获取的值是元素最后显示的值，不是最原始的值。

<html>
    <head>
    ...
    <style>
        #test{
            width: 500px;
            height: 500px;
            margin: auto;
        }
    </style>
    </head>
    <body>
        <div id="test"></div>
        <script>
            function $(element){
                return document.querySelector(element);
            }
            var test = $('#test');
            var style =  window.getComputedStyle(test,null);
            console.log(style.width);// 500px
            console.log(style.margin); // 0 702px
        </script>
    </body>
</html>

上面例子margin的值不是auto,而是元素当前css的值，jq的源码css()也不是用getComputedStyle()实现。

getComputed返回的是一个对象，要获取具体的值，可以用style.width来获取，getPropertyValue(‘properrtyName’)可以直接获取。

var style = window.getComputedStyle(test,null).getPropertyValue('width');
console.log(style);//500px
