科协的在线报名系统

三天前,在群里闲聊时,突然接到主席的任务要给院科协做一个在线报名系统,我随即大吃一惊,我心想我没弄过啊,问他去年有没有,他说他去年不会做,于是任务就到了我们的肩上了,就这样到了我和另一个小伙伴身上,庆幸的是我不用管服务器和数据库啦啦啦啦~~~

项目在GitHub上开源.

在线demo

技术栈(滑稽) html css javascript

项目分为五个页面

index.html 主页加了院科协的logo和注册和登录的按钮.

register.html 注册页面分为用户名,密码,确认密码,提交.

先写逻辑,再把相同功能的代码整合为一个函数

错误样式和正确样式

function correctStyle(e) {
    e.style.boxShadow = "0 0 0 0";
    e.style.borderColor = "#eee";
}
// 输入正确的样式
function errorStyle(e) {
    e.style.boxShadow = "0 0 8px 1px #eee";
    e.style.borderColor = "#e38383";
}
// 输入错误的样式

验证用户名,密码,确认密码,以确认密码为例

function testPassword_2(e) {
    var value_1 = password.value,
        value_2 = password_2.value;
    var array = value_2.toString().split('');
    if (array.length < 6) {
        errorStyle(e);
    } else if (array.length >= 6) {
        if (value_2 === value_1) {
            correctStyle(e);
            return true;
        } else {
            errorStyle(e);
        }
    }
}
// 验证两次输入的密码是否相等

在提交前验证,用onsubmit

register.onsubmit = function () {
    if (testUsername(username) && testPassword(password) && testPassword_2(password_2)) {
        sha1(username, password, password_2);
        return true;
    } else {
        alert('请正确输入');
        return false;
    }
};
// register提交时的验证

login.html页面大同小异,就不贴代码了,这些都是自己写的没有用插件,兼容性问题估计会爆炸.

signUp.html就是报名的页面了,登录后就会跳转到signUp页面.

tip.html页面就是报名成功后显示的状态页面,提示萌新们怎么联系我们.

这里用了两次正则表达式

验证用户名 

var reg = /^B{1}1[4|5|6|7]\d{6}$/;

验证手机号

var reg = /^1[3|4|5|7|8][0-9]\d{8}$/;
