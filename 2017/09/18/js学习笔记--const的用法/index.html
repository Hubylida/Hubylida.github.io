const用法

const定义一个常量

const a = 1;

const的值不能改变，const的作用域和let一样，只在块级作用域内有效。

const存在暂时性死区，可以认为存在变量提升，但是要等到执行完变量声明的语句后，调用才不会报错。

const和let一样不能重复声明变量。

说点别的

Object.freeze 方法

const foo = Object.freeze({});
//常规模式下，下一行代码不报错
//严格模式下，下一行代码报错
foo.name = "Yuan"

const保证的，并不是变量的值不能改动，而是变量指向的那个内存地处不能改动，不是变量的内存地址。

const foo = {};
foo.name = "Yuan"; // 不报错

foo = {} //报错
