js学习笔记--canvas方法

2018-03-25

方法

一个绘画状态包括：

当前应用的变形（即移动，旋转和缩放，见下）

strokeStyle, fillStyle, globalAlpha, lineWidth, lineCap, lineJoin, miterLimit, shadowOffsetX, shadowOffsetY, shadowBlur, shadowColor, globalCompositeOperation 的值当前的裁切路径（clipping path)你可以调用任意多次 save 方法。

每一次调用 restore 方法，上一个保存的状态就从栈中弹出，所有设定都恢复 transform(m11, m12, m21, m22, dx, dy)

基于自身参数的矩阵

m11 m21 dx

m12 m22 dy

0 0 1
m11 :水平方向的缩放

m12 :水平方向的偏移

m21 :竖直方向的偏移

m22 :竖直方向的缩放

dx :水平方向的移动

dy :竖直方向的移动

移动translate(dx,dy)

dx:水平方向上的偏移，dy:竖直方向上的偏移

缩放scale(sx,sy)

sx:水平方向上的缩放因子

sy:竖直方向上的缩放因子

## canvas大小

canvas有两个大小，一个是画布的大小，一个是画布在屏幕上的大小。

如果css设置的话，设置的只是画布在屏幕上的大小，画布大小和内容区域一起更改大小。

如果用js设置的话，

ctx.width = 100;

ctx.height = 100;

内容区域会消失，需要重新绘制。

因此更改大小时，把上述代码写在绘制代码之前，更改时就不需要重新绘制了。
