wepy开发问题汇总

2018-01-25

wepy开发问题汇总

  1. 不能新建空的wep文件，不然build后会一直报错

  2. import 组件时不能带.wpy

  3. 不支持background: url(‘’)

  4. 写路由时，pages前不能加斜杠,是绝对路径

  5. 写判断时，可以在data里写一个flag属性，data={ flag: true }

  6. 写menu-btn时，用到了wepy.createAnimation()方法创建了一个实例animatoin对象,再将animation使用方法。三个view组成三条斜杠，上面和下面用rotate方法，中间是用opcity方法实现三条杠变为叉号的动效，左边弹出的按钮使用translate实现

  7. 多个元素一个绑定类似的函数是用参数代替创建多个函数

  8. 实现中间向两边展开的动画是用margin: 0 50%; -> margin: 0 -50%

  9. 新建的wpy文件不仅不能为空而且template里面要有内容，不然不会生成wxml

  10. scroll-view要给定width，不然左右滑的时候页面会一起跟着滑

  11. 在函数运行周期之外的函数里去修改数据需要手动调用$apply方法

  12. repeat获取当前组件对index值可以用 this.$index获取当前组件对索引值

  13. 运用repeat时数组须为一维数组，不能为二维数组

  14. block这个标签没有意义，用于wx:if的判断

  15. dcard网站分析

  16. text-overflow: ellipsis;

  17. 导航和showToast不能一起用，应该设置定时器
    注意问题：加载快不快，容不容易推出小程序
